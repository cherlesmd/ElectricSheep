## Quick Facts
* Uses Java 17
* Built using Gradle
* Implements TarsosDSP for real time audio processing
* GUI created with JavaFX
* Makes use of multi-threading to seperate GUI and audio processing

## Download and Usage
Clone repo and run 
```bash
./gradlew run
```
